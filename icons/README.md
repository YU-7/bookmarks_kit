# 图标文件

请在此目录下放置以下尺寸的图标文件：
- `icon16.png` (16x16 像素) - 用于浏览器工具栏
- `icon48.png` (48x48 像素) - 用于扩展管理页面
- `icon128.png` (128x128 像素) - 用于 Chrome 网上应用店

这些图标将用于 Chrome 扩展的工具栏和扩展管理页面。

## 如何创建图标

你可以使用以下工具创建图标：
- [Favicon Generator](https://favicon.io/) - 在线图标生成器
- [Icon Generator](https://www.favicon-generator.org/) - 多尺寸图标生成
- 使用图像编辑软件（如 Photoshop、GIMP）创建并导出为 PNG 格式

**注意**：如果没有图标文件，扩展仍然可以运行，但会在 Chrome 中显示默认图标。
